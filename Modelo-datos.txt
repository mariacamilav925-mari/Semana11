MODELO DE DATOS - PLATAFORMA COMUNITARIA

Entidad: Causa
Campos principales:
- id
- titulo
- descripcion
- fecha

Propósito:
Guardar las causas comunitarias que serán visibles en la plataforma.

Entidad: Apoyo
Campos principales:
- id
- causa_id
- usuario
- fecha

Propósito:
Registrar los apoyos que recibe cada causa para hacer seguimiento a la participación.
